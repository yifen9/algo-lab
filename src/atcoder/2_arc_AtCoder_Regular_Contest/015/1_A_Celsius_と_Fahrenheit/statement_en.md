<div>

### **問題文**

<section>
摂氏での温度 $n$度を華氏にで表すと $( 9 ÷ 5 × n ) + 32$度となります。

摂氏での温度が与えられるので華氏での温度に変換するプログラムを書いてください。


</section>

</div>

---

<div>

### **入力**

<section>
入力は以下の形式で標準入力から与えられる。

<div>

$n$
</div>

<ul>

<li>
$1$行目は、華氏に変換すべき、摂氏の温度 $n\ (-273≦n≦1,000)$が整数で与えられる。
</li>

</ul>

</section>

</div>

<div>

### **出力**

<section>
摂氏での温度 $n$を華氏に変換し、その値を $1$行で出力せよ。

なお、出力の最後には改行をいれること。

出力は絶対誤差あるいは相対誤差の少なくとも片方が $10^{-6}$以下であれば許容される。

</section>

</div>

---

<div>

### **入力例 1**

<section>

```
10
```

</section>

</div>

<div>

### **出力例 1**

<section>

```
50
```

<ul>

<li>
$(9÷5×10)+32$$=$$50$
</li>

</ul>

</section>

</div>

---

<div>

### **入力例 2**

<section>

```
33
```

</section>

</div>

<div>

### **出力例 2**

<section>

```
91.4
```

<ul>

<li>
$(9÷5×33)+32$$=$$91.4$
</li>

<li>
摂氏での温度が整数であっても、華氏での温度は整数になるとは限りません。
</li>

</ul>

</section>

</div>

---

<div>

### **入力例 3**

<section>

```
-100
```

</section>

</div>

<div>

### **出力例 3**

<section>

```
-148
```

<ul>

<li>
$9÷5×(-100)$$+32$$=$$-148$
</li>

</ul>

</section>

</div>
