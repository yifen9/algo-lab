### **Problem Statement**

<p>
For an integer $x$, let $f(x)$be the number of leading ones in the decimal notation of $x$.
For example, we have $f(1)=1,f(2)=0,f(10)=1,f(11)=2,f(101)=1$.
</p>

<p>
Given an integer $N$, find $f(1)+f(2)+\cdots+f(N)$.
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$1 \leq N \leq 10^{15}$
</li>

<li>
All values in input are integers.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
Input is given from Standard Input in the following format:
</p>

<div>

$N$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Print the answer.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

11

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

4

</div>

<p>
We have $f(2)=f(3)=\cdots =f(9)=0$.
The answer is $f(1)+f(10)+f(11)=4$.
</p>

</section>
