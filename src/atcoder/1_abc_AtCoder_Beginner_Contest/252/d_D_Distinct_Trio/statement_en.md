<section>

### **Problem Statement**

<p>
You are given a sequence of length $N$: $A=(A_1,A_2,\ldots,A_N)$.

Find the number of triples $(i,j,k)$that satisfy both of the following conditions.
</p>

<ul>

<li>
$1\leq i \lt j \lt k \leq N$
</li>

<li>
$A_i$, $A_j$, and $A_k$are distinct.
</li>

</ul>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$3 \leq N \leq 2\times 10^5$
</li>

<li>
$1 \leq A_i \leq 2\times 10^5$
</li>

<li>
All values in input are integers.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
Input is given from Standard Input in the following format:
</p>

<div>

$N$$A_1$$A_2$$\ldots$$A_N$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Print the answer.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

4
3 1 4 1

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

2

</div>

<p>
The two triples $(i,j,k)$satisfying the conditions are $(1,2,3)$and $(1,3,4)$.
</p>

</section>
