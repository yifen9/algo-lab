### **Problem Statement**

<p>
You are given an integer $N$between $0$and $9999$(inclusive).
</p>

<p>
Print it as a four-digit string after appending to it the necessary number of leading zeros.
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$0 \leq N \leq 9999$
</li>

<li>
$N$is an integer.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
Input is given from Standard Input in the following format:
</p>

<div>

$N$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Print the answer.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

321

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

0321

</div>

<p>
$321$has three digits, so we need to add one leading zero to it to make it have four digits.
</p>

</section>

</div>
