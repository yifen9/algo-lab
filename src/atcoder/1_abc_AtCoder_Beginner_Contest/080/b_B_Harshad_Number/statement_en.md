### **Problem Statement**

<p>
An integer $X$is called a Harshad number if $X$is divisible by $f(X)$, where $f(X)$is the sum of the digits in $X$when written in base $10$.
</p>

<p>
Given an integer $N$, determine whether it is a Harshad number.
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$1?N?10^8$
</li>

<li>
$N$is an integer.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
Input is given from Standard Input in the following format:
</p>

<div>

$N$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Print `Yes`if $N$is a Harshad number; print `No`otherwise.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

12

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

Yes

</div>

<p>
$f(12)=1+2=3$. Since $12$is divisible by $3$, $12$is a Harshad number.
</p>

</section>

</div>

---

<div>

<section>

### **Sample Input 2**

<div>

57

</div>

</section>

</div>

<div>

<section>

### **Sample Output 2**

<div>

No

</div>

<p>
$f(57)=5+7=12$. Since $57$is not divisible by $12$, $12$is not a Harshad number.
</p>

</section>

</div>

---

<div>

<section>

### **Sample Input 3**

<div>

148

</div>

</section>

</div>

<div>

<section>

### **Sample Output 3**

<div>

No

</div>

</section>
