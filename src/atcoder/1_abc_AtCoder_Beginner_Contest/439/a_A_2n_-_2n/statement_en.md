### **Problem Statement**

<p>
You are given an integer $N$. Compute and output the value of $2^N-2N$.
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$N$is an integer between $1$and $11$, inclusive.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
The input is given from Standard Input in the following format:
</p>

<div>

$N$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Output the answer.
</p>

</section>

</div>

</div>

<div>

<section>

### **Sample Input 1**

1

</div>

</section>

</div>

<div>

0

</div>

</p>
