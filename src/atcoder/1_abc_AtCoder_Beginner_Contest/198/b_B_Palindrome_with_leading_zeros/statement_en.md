### **Problem Statement**

<p>
Given is an integer $N$.
</p>

<p>
Is it possible to add zero or more `0`s at the beginning of the string representing $N$in base ten to get a palindrome?
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$0 \leq N \leq 10^9$
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
Input is given from Standard Input in the following format:
</p>

<div>

$N$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
If a palindrome can be made, print `Yes`; otherwise, print `No`.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

1210

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

Yes

</div>

<p>
Adding one `0`at the beginning of `1210`results in `01210`, a palindrome.
</p>

</section>

</div>

---

<div>

<section>

### **Sample Input 2**

<div>

777

</div>

</section>

</div>

<div>

<section>

### **Sample Output 2**

<div>

Yes

</div>

<p>
`777`is already a palindrome.
</p>

</section>

</div>

---

<div>

<section>

### **Sample Input 3**

<div>

123456789

</div>

</section>

</div>

<div>

<section>

### **Sample Output 3**

<div>

No

</div>

</section>
