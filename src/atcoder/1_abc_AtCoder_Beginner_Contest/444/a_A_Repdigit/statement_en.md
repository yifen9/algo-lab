<section>

### **Problem Statement**

<p>
You are given a $3$-digit positive integer $N$.
Determine whether all digits are the same when $N$is represented in decimal.
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$100 \leq N \leq 999$
</li>

<li>
The input value is an integer.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
The input is given from Standard Input in the following format:
</p>

<div>

$N$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
If all digits are the same when $N$is represented in decimal, output `Yes`in one line; otherwise, output `No`.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

444

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

Yes

</div>

<p>
The digits of $444$are $4,4,4$, which are the same, so output `Yes`.
</p>

</section>

</div>

---

<div>

<section>

### **Sample Input 2**

<div>

160

</div>

</section>

</div>

<div>

<section>

### **Sample Output 2**

<div>

No

</div>

<p>
The digits of $160$are $1,6,0$, which are not the same, so output `No`.
</p>

</section>

</div>

---

<div>

<section>

### **Sample Input 3**

<div>

999

</div>

</section>

</div>

<div>

<section>

### **Sample Output 3**

<div>

Yes

</div>

</section>
