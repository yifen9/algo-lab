<section>

### **Problem Statement**

<p>
Find the number of 
<em>
palindromic numbers
</em>
among the integers between $A$and $B$(inclusive).
Here, a palindromic number is a positive integer whose string representation in base $10$(without leading zeros) reads the same forward and backward.
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$10000 \leq A \leq B \leq 99999$
</li>

<li>
All input values are integers.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
Input is given from Standard Input in the following format:
</p>

<div>

$A$$B$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Print the number of 
<em>
palindromic numbers
</em>
among the integers between $A$and $B$(inclusive).
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

11009 11332

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

4

</div>

<p>
There are four integers that satisfy the conditions: $11011$, $11111$, $11211$and $11311$.
</p>

</section>

</div>

31415 92653

</div>

</section>

</div>

<div>

<section>

612

</div>

</section>
