<section>

### **Problem Statement**

<p>
Takahashi is going to set a $3$-character password.
</p>

<p>
How many possible passwords are there if each of its characters must be a digit between $1$and $N$(inclusive)?
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$1 \leq N \leq 9$
</li>

<li>
$N$is an integer.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
Input is given from Standard Input in the following format:
</p>

<div>

$N$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Print the number of possible passwords.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

2

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

8

</div>

<p>
There are eight possible passwords: `111`, `112`, `121`, `122`, `211`, `212`, `221`, and `222`.
</p>

</section>

</div>

---

<div>

<section>

### **Sample Input 2**

<div>

1

</div>

</section>

</div>

<div>

<section>

### **Sample Output 2**

<div>

1

</div>

<p>
There is only one possible password if you can only use one kind of character.
</p>

</section>
