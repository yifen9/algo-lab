<section>

### **Problem Statement**

<p>
Find the number, modulo $998244353$, of permutations $P=(P_1,P_2,\dots,P_N)$of $(1,2,\dots,N)$such that:
</p>

<ul>

<li>
$|P_i - i| \ge X$for all integers $i$with $1 \le i \le N$.
</li>

</ul>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$1 \le N \le 100$
</li>

<li>
$1 \le X \le 5$
</li>

<li>
All input values are integers.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
The input is given from Standard Input in the following format:
</p>

<div>

$N$$X$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Print the answer.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

3 1

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

2

</div>

<p>
The conforming permutations $P=(P_1,P_2,P_3)$are the following two, $(2,3,1)$and $(3,1,2)$, so the answer is $2$.
</p>

</section>
