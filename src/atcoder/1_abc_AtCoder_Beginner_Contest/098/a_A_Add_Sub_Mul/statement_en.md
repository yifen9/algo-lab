<div>

<section>

### **Problem Statement**

<p>
You are given two integers $A$and $B$.
Find the largest value among $A+B$, $A-B$and $A \times B$.
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$-1000 \leq A,B \leq 1000$
</li>

<li>
All values in input are integers.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
Input is given from Standard Input in the following format:
</p>

<div>

$A$$B$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Print the largest value among $A+B$, $A-B$and $A \times B$.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

3 1

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

4

</div>

<p>
$3+1=4$, $3-1=2$and $3 \times 1=3$. The largest among them is $4$.
</p>

</section>

</div>

---

<div>

<section>

### **Sample Input 2**

<div>

4 -2

</div>

</section>

</div>

<div>

<section>

### **Sample Output 2**

<div>

6

</div>

<p>
The largest is $4 - (-2) = 6$.
</p>

</section>

</div>

---

<div>

<section>

### **Sample Input 3**

<div>

0 0

</div>

</section>

</div>

<div>

<section>

### **Sample Output 3**

<div>

0

</div>

</section>
