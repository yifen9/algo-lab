<section>

### **Problem Statement**

<p>
How many positive integers no greater than $N$can be represented as $a^2 \times b \times c^2$with three 
<strong>
primes
</strong>
$a,b$, and $c$such that $a<b<c$?
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$N$is an integer satisfying $300 \le N \le 10^{12}$.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
The input is given from Standard Input in the following format:
</p>

<div>

$N$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Print the answer as an integer.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

1000

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

3

</div>

<p>
The conforming integers no greater than $1000$are the following three.
</p>

<ul>

<li>
$300 = 2^2 \times 3 \times 5^2$
</li>

<li>
$588 = 2^2 \times 3 \times 7^2$
</li>

<li>
$980 = 2^2 \times 5 \times 7^2$
</li>

</ul>

</section>

</div>

---

<div>

1000000000000

</div>

</section>

</div>

<div>

<section>

### **Sample Output 2**

<div>

2817785

</div>

</section>
