<section>

### **Problem Statement**

<p>
There are $N$seats arranged in a row. Each seat can accommodate at most one person. Determine whether it is possible to seat $M$people in the seats satisfying the following condition.
</p>

<blockquote>

<p>
No two adjacent seats may both be occupied.
</p>

</blockquote>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$1 \le N, M \le 100$
</li>

<li>
All input values are integers.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
The input is given from Standard Input in the following format:
</p>

<div>

$N$$M$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Output `Yes`if it is possible to seat the people satisfying the condition, and `No`otherwise.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

6 3

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

Yes

</div>

<p>
For example, seating people in the first, third, and sixth seats from the left is one valid way. Thus, output `Yes`.
</p>

</section>

</div>

---

<div>

<section>

### **Sample Input 2**

<div>

4 3

</div>

</section>

</div>

<div>

<section>

### **Sample Output 2**

<div>

No

</div>

<p>
There is no valid way to seat the people satisfying the condition. Thus, output `No`.
</p>

</section>

</div>

---

<div>

<section>

### **Sample Input 3**

<div>

5 3

</div>

</section>

</div>

<div>

<section>

### **Sample Output 3**

<div>

Yes

</div>

</section>

</div>

---

<div>

<section>

### **Sample Input 4**

<div>

44 7

</div>

</section>

</div>

<div>

<section>

### **Sample Output 4**

<div>

Yes

</div>

</section>
