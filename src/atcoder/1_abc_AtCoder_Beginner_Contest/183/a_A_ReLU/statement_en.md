### **Problem Statement**

<p>
The function ReLU is defined as follows:
<a href="https://atcoder.jp/contests/abc183/tasks/abc183_aReLU(x) =
        \begin{cases}
            x & (x \geqq 0) \
            0 & (x < 0) \
        \end{cases}"></a>
</p>

<p>

<img src="https://img.atcoder.jp/ghi/relu.png">

</img>

</p>

<p>
Given an integer $x$, find $ReLU(x)$.
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$x$is an integer.
</li>

<li>
$-10 \leq x \leq 10$
</li>

</ul>

</section>

</div>

</section>

</div>

<div>

<section>

</p>

</section>

</div>

</div>
