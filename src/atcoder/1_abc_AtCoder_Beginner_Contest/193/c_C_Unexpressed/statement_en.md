<section>

### **Problem Statement**

<p>
Given is an integer $N$. How many integers between $1$and $N$(inclusive) are unrepresentable as $a^b$, where $a$and $b$are integers not less than $2$?
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$N$is an integer.
</li>

<li>
$1 ≤ N ≤ 10^{10}$
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
Input is given from Standard Input in the following format:
</p>

<div>

$N$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Print the answer.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

8

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

6

</div>

<p>
$4$and $8$are representable as $a^b$: we have $2^2 = 4$and $2^3 = 8$.

On the other hand, $1$, $2$, $3$, $5$, $6$, and $7$are unrepresentable as $a^b$using integers $a$and $b$not less than $2$, so the answer is $6$.
</p>

</section>

</div>

</section>

</div>

<div>

<section>

### **Sample Output 2**

99634

</div>

</section>
