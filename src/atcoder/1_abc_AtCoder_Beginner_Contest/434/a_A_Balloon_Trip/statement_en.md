<section>

### **Problem Statement**

<p>
Given integers $W$and $B$, solve the following problem.
</p>

<p>
Takahashi's weight is $W {\rm [kg]}$. (Note that the unit is ${\rm kg}$.)

An object attached with $n$balloons will fly into the sky if and only if the mass of the object is 
<strong>
strictly less than
</strong>
$nB$${\rm [g]}$.

What is the minimum number of balloons needed to make Takahashi fly into the sky?
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$1 \le W \le 100$
</li>

<li>
$1 \le B \le 100$
</li>

<li>
All input values are integers.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
The input is given from Standard Input in the following format:
</p>

<div>

$W$$B$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Output the answer.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

80 5

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

16001

</div>

<p>
Takahashi's weight is $80$${\rm kg}$$=$$80000$${\rm g}$.

If he is attached with $16001$balloons, his mass is less than $16001 \times 5=80005$${\rm g}$, so he will fly into the sky.

Note that $16000$balloons are not sufficient.
</p>

</section>

</div>
