<section>

### **Problem Statement**

<p>
How many integers $x$between $1$and $N$, inclusive, can be expressed as $x = a^b$using some positive integer $a$and a positive integer $b$
<strong>
not less than $2$
</strong>
?
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
All input values are integers.
</li>

<li>
$1 \le N \le 10^{18}$
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
The input is given from Standard Input in the following format:
</p>

<div>

$N$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Print the answer as an integer.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

99

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

12

</div>

<p>
The integers that satisfy the conditions in the problem statement are $1, 4, 8, 9, 16, 25, 27, 32, 36, 49, 64, 81$: there are $12$.
</p>

</section>

</div>
