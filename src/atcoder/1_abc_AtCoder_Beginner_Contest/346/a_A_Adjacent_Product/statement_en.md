<p>
Score: $100$points
</p>

<div>

<section>

### **Problem Statement**

<p>
You are given $N$integers $A_1, A_2, \dots, A_N$.
Also, define $B_i = A_i \times A_{i+1}\ (1 \leq i \leq N-1)$.
</p>

<p>
Print $B_1, B_2, \dots, B_{N-1}$in this order, separated by spaces.
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$2 \leq N \leq 100$
</li>

<li>
$1 \leq A_i \leq 100$
</li>

<li>
All input values are integers.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
The input is given from Standard Input in the following format:
</p>

<div>

$N$$A_1$$A_2$$\dots$$A_N$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Print $B_1, B_2, \dots, B_{N-1}$in this order, separated by spaces.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

3
3 4 6

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

12 24

</div>

<p>
We have $B_1 = A_1 \times A_2 = 12, B_2 = A_2 \times A_3 = 24$.
</p>

</section>

</div>
