<div>

<section>

### **Problem Statement**

<p>
Print a number between $A$and $B$(inclusive) that is a multiple of $C$.
</p>

<p>
If there is no such number, print `-1`.
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$1 \leq A \leq B \leq 1000$
</li>

<li>
$1 \leq C \leq 1000$
</li>

<li>
All values in input are integers.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
Input is given from Standard Input in the following format:
</p>

<div>

$A$$B$$C$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Print the answer.

If there is no number with the desired property, print `-1`.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

123 456 100

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

200

</div>

<p>
`300`or `400`would also be accepted.
</p>

</section>

</div>

---

<div>

<section>

### **Sample Input 2**

<div>

630 940 314

</div>

</section>

</div>

<div>

<section>

### **Sample Output 2**

<div>

-1

</div>

</section>
