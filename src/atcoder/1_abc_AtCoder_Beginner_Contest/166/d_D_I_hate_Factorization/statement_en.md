<div>

<section>

### **Problem Statement**

<p>
Give a pair of integers $(A, B)$such that $A^5-B^5 = X$.
It is guaranteed that there exists such a pair for the given integer $X$.
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$1 \leq X \leq 10^9$
</li>

<li>
$X$is an integer.
</li>

<li>
There exists a pair of integers $(A, B)$satisfying the condition in Problem Statement.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
Input is given from Standard Input in the following format:
</p>

<div>

$X$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Print $A$and $B$, with space in between.
If there are multiple pairs of integers $(A, B)$satisfying the condition, you may print any of them.
</p>

<div>

$A$$B$
</div>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

33

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

2 -1

</div>

<p>
For $A=2$and $B=-1$, $A^5-B^5 = 33$.
</p>

</section>

</div>

---

<div>

<section>

### **Sample Input 2**

<div>

1

</div>

</section>

</div>

<div>

<section>

### **Sample Output 2**

<div>

0 -1

</div>

</section>
