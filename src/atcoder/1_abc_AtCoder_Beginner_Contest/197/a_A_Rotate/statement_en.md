<section>

### **Problem Statement**

<p>
Given is a string $S$of length $3$.

Move the first character of $S$to the end of $S$and print the resulting string $S'$.
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$S$is a string of length $3$consisting of lowercase English letters.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
Input is given from Standard Input in the following format:
</p>

<div>

$S$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Print $S'$.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

abc

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

bca

</div>

<p>
Moving the first character `a`of the string `abc`results in `bca`.
</p>

</section>

</div>

aab

</div>

</section>

<div>

aba

</div>
