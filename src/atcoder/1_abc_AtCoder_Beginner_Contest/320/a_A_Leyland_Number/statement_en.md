<section>

### **Problem Statement**

<p>
You are given positive integers $A$and $B$.

Print the value $A^B+B^A$.
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$2 \leq A \leq B \leq 9$
</li>

<li>
All input values are integers.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
The input is given from Standard Input in the following format:
</p>

<div>

$A$$B$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Print the answer as an integer.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

2 8

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

320

</div>

<p>
For $A = 2, B = 8$, we have $A^B = 256, B^A = 64$, so $A^B + B^A = 320$.
</p>

</section>

</div>
