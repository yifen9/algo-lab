### **Problem Statement**

<p>
You are given an integer $N$.
If $N$is between $-2^{31}$and $2^{31}-1$(inclusive), print `Yes`; otherwise, print `No`.
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$-2^{63} \leq N < 2^{63}$
</li>

<li>
$N$is an integer.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
Input is given from Standard Input in the following format:
</p>

<div>

$N$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
If $N$is between $-2^{31}$and $2^{31}-1$(inclusive), print `Yes`; otherwise, print `No`.
</p>

</section>

</div>

</div>
