### **Problem Statement**

<p>
You are given a string $S$consisting of lowercase English letters.
</p>

<p>
Output the string obtained by appending `s`to the end of the string $S$.
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$S$is a string of length between $1$and $10$, inclusive, consisting of lowercase English letters.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
The input is given from Standard Input in the following format:
</p>

<div>

$S$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Output the answer.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

http

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

https

</div>

<p>
Appending `s`to the end of `http`results in `https`, so print `https`.
</p>
