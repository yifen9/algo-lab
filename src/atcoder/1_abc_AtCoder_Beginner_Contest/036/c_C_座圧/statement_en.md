<section>

### **問題文**
$N$人の人が座っています。
$i$番目の人の座圧は $a_i$です。

すぬけ君は、大小関係を保存したまま座圧のデータを圧縮して保存することにしました。
以下の条件を満たす数列 $b_1, …, b_N$を求めてください。


<ul>

<li>
$b_i$はすべて非負整数である。
</li>

<li>
$a_i < a_j$ならば $b_i < b_j$である。
</li>

<li>
$a_i = a_j$ならば $b_i = b_j$である。
</li>

<li>
上の条件を満たす配列のうち、$b_i$の最大値が最小となる。
</li>

</ul>
このような条件をみたす $b$は一意に定まることが知られています。


</section>

<section>

### **制約**

<ul>

<li>
$1 ≤ N ≤ 10^5$
</li>

<li>
$0 ≤ a_i ≤ 10^9$
</li>

<li>
$a_i$は整数である。
</li>

</ul>

</section>

<section>

### **部分点**

<ul>

<li>
$30$点分のテストケースでは、$1 ≤ N ≤ 10^3$をみたす。
</li>

<li>
上とは別の $30$点分のテストケースでは、$0 ≤ a_i ≤ 10^5$をみたす。
</li>

</ul>

</section>

---

<div>

<section>

### **入力**

<p>
入力は以下の形式で標準入力から与えられる。
</p>

<div>

$N$$a_1$$:$$a_N$
</div>

</section>

<section>

### **出力**
$N$行出力せよ。$i$行目には $b_i$を出力せよ。

</section>

</div>

</div>
