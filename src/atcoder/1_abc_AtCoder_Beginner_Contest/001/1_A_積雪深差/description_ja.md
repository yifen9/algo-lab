### **問題文**
<section>

ある時刻の積雪深 $H_1$ と その $1$ 時間前の積雪深 $H_2$ が与えられます。この時、この $1$ 時間の積雪深差 $H_1$$-$$H_2$ の値を計算して出力してください。

</section>
</div>
---
<div>
<div>
### **入力**
<section>

入力は以下の形式で標準入力から与えられる。


$H_1$
$H_2$


<ol>
<li>
$1$ 行目には、整数で、ある時刻の積雪深 $H_1\ (0≦H_1≦2,000)$ が与えられる。</li>
<li>
$2$ 行目には、整数で、$1$ 時間前の積雪深 $H_2\ (0≦H_2≦2,000)$ が与えられる。</li>
</ol>
</section>
</div>
<div>
### **出力**
<section>

積雪深差 $H_1$$-$$H_2$ の値を $1$ 行で出力せよ。

また、出力の末尾には改行を入れること。
</section>
</div>
</div>
---
<div>
### **入力例 1**
<section>
```
15
10
```
</section>
</div>
<div>
### **出力例 1**
<section>
```
5
```
<ul>
<li>
$15-10$$=5$</li>
</ul>
</section>
</div>
---
<div>
### **入力例 2**
<section>
```
0
0
```
</section>
</div>
<div>
### **出力例 2**
<section>
```
0
```
<ul>
<li>
$0-0$$=0$</li>
</ul>
</section>
</div>
---
<div>
### **入力例 3**
<section>
```
5
20
```
</section>
</div>
<div>
### **出力例 3**
<section>
```
-15
```
<ul>
<li>
$5-20$$=-15$</li>
</ul>
</section>
