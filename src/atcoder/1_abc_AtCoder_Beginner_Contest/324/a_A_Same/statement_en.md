
<div>

<span>

<span>

<p>
Score : $100$points
</p>

<div>

<section>

### **Problem Statement**

<p>
You are given $N$integers $A _ 1,A _ 2,\ldots,A _ N$.
</p>

<p>
If their values are all equal, print `Yes`; otherwise, print `No`.
</p>

</section>

</div>

<div>

<section>

### **Constraints**

<ul>

<li>
$2\leq N\leq100$
</li>

<li>
$1\leq A _ i\leq100\ (1\leq i\leq N)$
</li>

<li>
All input values are integers.
</li>

</ul>

</section>

</div>

---

<div>

<div>

<section>

### **Input**

<p>
The input is given from Standard Input in the following format:
</p>

<div>

$N$$A _ 1$$A _ 2$$\ldots$$A _ N$
</div>

</section>

</div>

<div>

<section>

### **Output**

<p>
Print a single line containing `Yes`if the values of the given $A _ 1,A _ 2,\ldots,A _ N$are all equal, and `No`otherwise.
</p>

</section>

</div>

</div>

---

<div>

<section>

### **Sample Input 1**

<div>

3
3 2 4

</div>

</section>

</div>

<div>

<section>

### **Sample Output 1**

<div>

No

</div>

<p>
We have $A _ 1\neq A _ 2$, so you should print `No`.
</p>

</section>

</div>

---

<div>

<section>

### **Sample Input 2**

<div>

4
3 3 3 3

</div>

</section>

</div>

<div>

<section>

### **Sample Output 2**

<div>

Yes

</div>

<p>
We have $A _ 1=A _ 2=A _ 3=A _ 4$, so you should print `Yes`.
</p>

</section>

</div>

---

<div>

<section>

### **Sample Input 3**

<div>

10
73 8 55 26 97 48 37 47 35 55

</div>

</section>

</div>

<div>

<section>

### **Sample Output 3**

<div>

No

</div>

</section>
